# Command Line Arguments Parser

This app parses arguments given on CLI.

## Run Command Line Parser

```
cd target
cd debug
./cli_args_parser arg1 arg2 "arg3 in exclamation marks"
```

## Learnings

Macros like ```println?``` are marked with a ```?```.
